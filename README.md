<h1>👨‍💻 Primera Administración de Base de Datos - Zapatería Par2</h1>

<h2>📌 Descripción del proyecto</h2>

<p>
Hola 👋 en este repositorio se encuentra una base de datos de una zapatería, la cual cuenta con 5 tablas. Cuatro de ellas tienen 25 registros cada una, mientras que la tabla de empleados contiene 5 registros, esto debido a que la zapatería Par2 no cuenta con 25 empleados.
</p>

<p>
Cada una de las tablas tiene sus respectivas relaciones para asegurar su buen funcionamiento dentro de la base de datos.
</p>

<p>
Este proyecto fue realizado como práctica utilizando MySQL y phpMyAdmin.
</p>
